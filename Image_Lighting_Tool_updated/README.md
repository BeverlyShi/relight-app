
  # Image Lighting Tool

  This is a code bundle for Image Lighting Tool. The original project is available at https://www.figma.com/design/Pkw4cC353QlbYxxLqYleRR/Image-Lighting-Tool.

  ## Running the code

  Run `npm i` to install the dependencies.

  Copy `.env.example` to `.env` and set backend URL if needed:
  - `VITE_RELIGHT_API_BASE_URL=http://localhost:6007`

  Run `npm run dev` to start the development server.
